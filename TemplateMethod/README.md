# TemplateMethodパターン
## パターンの用途メリットなど
全体のフローが固定だけど具体的な処理が場合によって全く違う場合に使える
## サンプルの説明-照明管理システム
時間によって照明のオンオフと朝昼晩に割り当てられた照明の処理を行うシステムです。
照明のスケジュールが固定で、具体的な照明のセッティングはサブクラスに任せています。
