# Bridgeパターン
## 適用パターン
変更や追加が予想される要素だけを切り離して、それぞれを独立して拡張できる

## サンプル 試合の得点カウンター
試合の得点をカウントするシステムです。
抽象的なカウンターは勝ちか負けかだけを、実装のほうはスポーツの種目によってカウントの増え方や呼び方の違いを実装します。
抽象的なカウンターは具体的な実装を利用して得点を表示、管理などの機能追加ができます。
