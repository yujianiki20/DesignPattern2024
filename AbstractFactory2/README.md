# AbstractFactory
# 用途など
インスタンスを構成する抽象的な意味での部品が共通で最終的に作成されるものが
ファクトリーごとに統一された規格の部品で作られなければいけないものに適している。

## 課題
ゲームコントローラーの定義システムです
抽象的な部品としてボタンの役割をクラスにして、それを継承して実際の役割を持たせています。
